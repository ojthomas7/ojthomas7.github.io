Overview

Datasets within astronomy can grow to be incredibly large. Until data is annotated, it is incredibly difficult for it to be put to use. The idea behind this project was to use knowledge gained within the codecademy online course "Creating deep learning models using TensorFlow" to create a machine learning model that analyses large datasets of images of galaxies and sorts them into four categories: Lenticular, Galaxies with rings, Galactic mergers and irregular.

This was achieved by writing code that takes an input of a database of many images of different galaxies, prepares the data by splitting into training and validation sets, creates a machine learning model, and trains this model over 8 epochs. The end result is a classification of the supplied images into one of four categories with roughly 61% accuracy.
