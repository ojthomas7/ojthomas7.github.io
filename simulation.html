Overview

This project formed the second half of my dissertation module. Rayleigh-Benard convection is a form of natural convection that occurs when a fluid is heated from below and cooled from above. This process is driven by buoyancy forces arising from temperature-induced density variations in the fluid, and within this project I simulate these processes within Python. Specifically I utilise the DEDALUS framework, used to solve complex PDEs. This included writing code to construct instances of convection, beginning with 2-D Rayleigh-Benard convection and extending this into astrophysics by considering the convection within a rotating spherical shell. The physical relevance of this extension being comparison between this shell and the convection zone in stars.
