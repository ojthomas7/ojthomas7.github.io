Overview

The objective of this project was to write an automated python script that investigates the financial performance of the publicly traded company "Palantir Technologies", specifically their quarterly revenue over time. This was completed via the following steps:

  • Web navigation with selenium: navigating to the relevant investor relations page, dealing with any pop-ups, and selecting the quarterly revenue results
  • Data extraction with BeautifulSoup: using BeautifulSoup to parse the HTML data, identify and extract the relevant figures and corresponding dates
  • Data processing: cleaning the extracted data, removing abnormalities and converting to the correct data type
  • Visualising the data
