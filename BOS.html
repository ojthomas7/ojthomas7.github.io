Overview

This project formed the first half of my final year dissertation module. Schlieren imaging is a method of visualising fluctuations in the density of a transparent medium, used within engineering for investigating shockwave interactions and aerodynamic boundary layers. A classical schlieren setup requires a controlled environment and heavy, expensive equipment. In my investigation I decided to investigate background oriented schlieren (BOS), a schlieren method that relies only on software. I developed my own BOS setup within python using the computer vision module OpenCV. To the right is an example of my code in action, visualising the shockwaves created by super-sonic jets flying in formation.
